## 1. CHIP 基础知识框架

### 1.1 定义与诊断标准

| 标准 | 内容 |
|------|------|
| **定义** | 外周血或骨髓中存在体细胞突变（驱动基因），VAF≥2%，但不符合血液恶性肿瘤诊断标准 |
| **主要突变基因** | DNMT3A (~50%), TET2 (~20%), ASXL1 (~10%), JAK2, TP53, SF3B1 等 |
| **流行率** | <40岁 <1%, 40-60岁 ~5%, >70岁 >10%, >85岁 >20% |
| **女性更年期** | 女性绝经后CHIP流行率加速上升，DNMT3A突变在女性中富集 |

### 1.2 突变类型与功能后果

| 基因 | 功能 | 突变效应 | 已知疾病关联 |
|------|------|---------|-------------|
| **DNMT3A** | DNA甲基转移酶（添加甲基到CpG岛） | 功能缺失 → 低甲基化 → 染色质开放 → 炎症转录因子激活 | CVD, 骨质疏松, 牙周炎, OA, RA |
| **TET2** | DNA去甲基化酶（氧化5mC→5hmC） | 功能缺失 → 高甲基化 → NLRP3/IL-1β通路激活 | CVD, RA（血清阴性） |
| **ASXL1** | 染色质修饰复合物组分 | 功能缺失 → H2AK119去泛素化异常 | 骨髓增生异常 |

*来源: Jaiswal & Ebert, Science 2019; Jaiswal et al., NEJM 2017*

## 2. 核心证据：CHIP → 炎性骨丢失

**Wang et al., *Cell* 2024** — 本文是本项目的最核心基石

### 2.1 人群证据（ARIC队列, n=4,946）

| 结果 | 数据 | 显著性 |
|------|------|--------|
| Stage IV牙周炎 | DNMT3A-CHIP携带者 vs 非携带者: AME +11 pp | 趋势p=0.017 |
| 临床附着丧失(CAL) | 29.3% vs 23.9%位点 ≥3mm | p=0.017 |
| 龈炎指数≥1 | 43.4% vs 34.7%位点 | p=0.027 |

### 2.2 动物模型机制（核心）

**模型**: 10% Dnmt3aR878H/+ BM细胞 + 90% WT BM，竞争性移植（模拟人CHIP克隆大小）

```
Dnmt3aR878H/+ 突变（人R882H热点突变同源）
    │
    ├──→ 髓系偏向：CD11b+ 单核/巨噬细胞↑
    │       └── 骨髄破骨前体细胞↑ (OC precursors)
    │       └── 外周破骨细胞生成性巨噬细胞↑ (OCM: Ly6CintI-A+E+CX3CR1hiF4/80+)
    │
    ├──→ 淋巴系：CD3+ T细胞↑
    │       └── Th17/Treg失衡 → IL-17↑, Treg功能↓
    │
    ├──→ 中性粒细胞：数量↑ + 功能亢进（ROS↑, IL-1β↑, IL-6↑, TNF↑）
    │
    └──→ 旁分泌效应：WT细胞在突变环境中也变成促炎表型
            ↓
    IL-17依赖性炎症 + 破骨细胞生成↑
            ↓
    自然发生和实验诱导的牙周炎↑ + 关节炎(CAIA)↑
```

**治疗**: 雷帕霉素(mTOR抑制剂) → 抑制CHIP克隆扩展 + 牙周炎

### 2.3 对本项目的意义

> 🔥 **DNMT3A-CHIP同时导致牙周炎和关节炎(CAIA)的骨破坏**
> **→ 证明CHIP驱动的炎性骨丢失是泛关节性的，非牙周组织特异性**
> **→ TMJ髁突软骨下骨同样可能受此通路影响**

## 3. 核心证据：CHIP → 骨质疏松

**Kim et al., *JEM* 2021** — Ebert实验室（CHIP奠基团队）

### 3.1 人群证据（UK Biobank, n=113,641）

| 关联 | 效应 |
|------|------|
| CHIP → 新发骨质疏松诊断 | 显著增加 |
| CHIP → 骨密度(eBMD) | 显著降低 |
| VAF≥10%的CHIP | eBMD降低更大 |

### 3.2 分子机制

```
Dnmt3a-/- 巨噬细胞
    ↓
DNA低甲基化 → 染色质开放 → Irf3-NF-κB激活
    ↓
IL-20 分泌 ↑（旁分泌）
    ↓
IL-20受体(IL20ra) → 破骨前体细胞分化 ↑
    ↓
破骨细胞生成↑ → 骨丢失
```

**治疗**: 阿仑膦酸钠(alendronate) / IL-20中和抗体 → 可逆转

### 3.3 关键差异点

| 方面 | Wang 2024 (Cell) | Kim 2021 (JEM) |
|------|-----------------|----------------|
| 模型 | 10%竞争移植(模拟人CHIP VAF) | 100% Dnmt3a-/- BM移植 |
| 主要因子 | IL-17 (T细胞/中性粒细胞) | IL-20 (巨噬细胞) |
| 效应细胞 | 中性粒细胞/OCM/Th17 | 巨噬细胞/IL20ra+破骨前体 |
| **共同通路** | **RANKL/破骨细胞生成↑, Treg功能障碍** | **Irf3-NF-κB炎症通路** |

> ⚡ 两个独立团队，两种模型，**一致结论**：DNMT3A-CHIP → 促炎 → 破骨细胞生成↑ → 骨丢失
> **这个共识是非常坚实的**

## 4. 核心证据：更年期 → FBM → DNMT3A优势

**Zioni et al., *Nat Commun* 2023** — 修正：第一作者Zioni非Koh

### 4.1 关键发现

| 发现 | 细节 |
|------|------|
| **FBM选择性优势** | 脂肪骨髓(FBM)为DNMT3A突变HSPCs提供选择性自我更新优势 |
| **年龄效应** | 老年小鼠的DNMT3AMut-HSPC在与FBM相互作用时优势更强 |
| **分子机制** | scRNA-seq → IFN-γ + IL-6炎症信号通路激活 |
| **治疗靶点** | 抗IL-6中和抗体可逆（FBM中的DNMT3A优势） |
| **模型** | 去势(castration)雄性小鼠 → 模拟性激素缺乏 |

### 4.2 更年期女性特异性的完整循环

```
更年期 → 雌激素骤降
     ↓
骨髓脂肪化(FBM)急剧增加（女性远快于男性）
     ↓
FBM旁分泌IL-6/IFN-γ信号 → DNMT3A突变选择性优势
     ↓
DNMT3A-CHIP克隆扩增
     ↓
促炎巨噬细胞/中性粒细胞/T细胞
     ↓
IL-17/IL-20/IL-1β/IL-6释放 ↑
     ↓
系统性炎症 + 破骨细胞生成↑
     ↓
TMJ软骨下骨丢失 + 骨破坏
     ↓
TMJOA加重 ←── 恶性循环
```
