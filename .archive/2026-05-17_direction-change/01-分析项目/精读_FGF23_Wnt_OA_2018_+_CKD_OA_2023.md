## 文献一：FGF23调控Wnt/β-catenin信号介导的小鼠OA

| 字段 | 内容 |
|------|------|
| **PMID** | 29718273 |
| **Title** | FGF23 regulates Wnt/β-catenin signaling-mediated osteoarthritis in mice overexpressing high molecular weight FGF2 |
| **Authors** | Patience Meo Burt, Liping Xiao, Marja M Hurley |
| **Journal** | Endocrinology |
| **Year** | 2018 |
| **DOI** | 10.1210/en.2018-00184 |
| **Institution** | UConn Health, Farmington, CT |

### 研究背景

- XLH（X连锁低磷血症）和Hyp小鼠（XLH同源模型）已知会发生退行性关节病
- 但驱动OA表型的确切机制不清楚
- HMWTg小鼠（过表达高分子量FGF2）表型复制XLH/Hyp，包括OA、血清FGF23升高

### 核心发现

#### 1. HMWTg关节Wnt通路激活（体内）

| 分子 | HMWTg vs Vector | 意义 |
|------|----------------|------|
| **Sost** | **消失** | Wnt抑制解除 |
| **Dkk1** | ↓ | Wnt抑制解除 |
| **Lrp6** | ↓ | 受体变化 |
| **Lrp5** | **↑↑** | LRP5是Wnt诱导软骨破坏的关键 |
| **Wnt5a** | ↑ | 非经典Wnt |
| **Wnt7b** | **↑↑** | **纯经典Wnt配体** |
| **p-GSK3β** | ↑ | GSK3β失活→β-catenin稳定 |
| **LEF1** | ↑ | 下游转录因子 |
| **核β-catenin** | **仅在HMWTg出现** | **经典Wnt激活金标准** |
| **Axin2** | ↑ | 靶基因 |

#### 2. HMWTg软骨细胞表型（体外）

| 指标 | HMWTg vs Vector | LMWTg |
|------|----------------|-------|
| Alcian blue | ↑↑（GAG增加） | 略↓ |
| ALP染色 | **↑↑（矿化/肥大）** | **↓↓** |
| **Fgf23 mRNA** | **↑↑** | ↓ |
| **Adamts5** | **↑↑** | — |
| **Il-1β** | **↑↑** | ↓ |
| **Wnt7b** | **↑↑** | ↓ |
| **Wnt16** | **↑↑** | — |
| **Wisp1** | **↑↑** | — |
| p-GSK3β蛋白 | **↑↑** | — |

**关键**：LMWTg小鼠被保护免受OA，与其形成鲜明对比。

#### 3. Wnt抑制剂IWR-1挽救实验

| 指标 | IWR-1处理后 |
|------|-------------|
| Axin2蛋白 | ↑（确认药物起效） |
| 活性β-catenin/总β-catenin | ↓ |
| Alcian blue | HMWTg降至Vector水平 |
| ALP染色 | HMWTg降至Vector水平 |
| **Fgf23 mRNA** | **HMWTg显著↓** |
| **Adamts5** | **HMWTg降至正常** |
| Wnt7b | HMWTg↓ |

**推论**：Wnt/β-catenin → FGF23正反馈环（β-catenin激活FGF23启动子）。

#### 4. FGF23中和抗体体内挽救（**首次**）

| 指标 | HMWTg + FGF23Ab vs HMWTg + Vehicle |
|------|-----------------------------------|
| 数字X线 | 胫骨平台扁平、关节间隙狭窄 → **恢复** |
| 3D µCT | 软骨下骨轮廓异常/侵蚀 → **恢复正常** |
| µCT骨参数 | BV/TV↑、Tb.Th↑、Tb.N↑、Tb.Sp↓ → **部分正常化** |
| **ColX mRNA** | **↓**（肥大标志） |
| **Mmp13 mRNA** | **↓** |
| **Mmp9 mRNA** | **↓** |
| **MMP13蛋白** | **消失** |
| **MMP9蛋白** | **消失** |
| **软骨厚度** | **恢复至Vector水平** |

### 核心机制图

```
HMW FGF2过表达
    ↓
骨和血清FGF23↑↑
    ↓
关节软骨FGF23↑ + FGFR1↑（Klotho非依赖）
    ↓
    ├─ Sost↓（Wnt抑制解除）
    ├─ Dkk1↓（Wnt抑制解除）
    ├─ Lrp5↑（促破坏受体增加）
    ├─ Lrp6↓
    ├─ Wnt7b↑（纯经典配体）
    ├─ Wnt5a↑（可激活经典通路）
    └─ p-GSK3β↑ → β-catenin稳定 → 核转位
        ↓
        LEF1↑ + Wisp1↑ + Axin2↑
        ↓
    ├─ Adamts5↑（蛋白聚糖降解）
    ├─ Mmp13↑（胶原降解）
    ├─ Mmp9↑
    ├─ ColX↑（肥大分化）
    ├─ Il-1β↑（炎症）
    └─ FGF23↑（正反馈）
        ↓
    软骨细胞加速分化/肥大 → ECM降解 → OA
```

### 治疗转化意义

| 发现 | 转化价值 |
|------|---------|
| FGF23Ab部分挽救OA | **FGF23中和抗体是潜在DMOAD** |
| IWR-1挽救肥大表型 | Wnt抑制剂可能有用（但全身毒性） |
| FGF23-Wnt正反馈环 | 打断任一节点均可阻断 |
| LMW FGF2保护OA | FGF2亚型平衡是内源性保护机制 |

### 局限性

- 仅使用HMWTg模型（XLH模拟），非自然CKD模型
- FGF23Ab剂量和时机未优化
- 未检测血清FGF23水平与关节病变严重度的相关性
- 无软骨下骨-软骨交互作用的直接证据

## 文献二：CKD-OA共病模型的双向疾病修饰作用

| 字段 | 内容 |
|------|------|
| **PMID** | 37562858 |
| **Title** | Disease-modifying interactions between chronic kidney disease and osteoarthritis: a new comorbid mouse model |
| **Authors** | Julovi SM, Dao A, Trinh K, et al. |
| **Journal** | RMD Open |
| **Year** | 2023 |
| **DOI** | 10.1136/rmdopen-2023-003109 |
| **Institution** | University of Sydney, Westmead Hospital |

### 研究设计

- **模型**：5/6肾切除（5/6Nx）± DMM（内侧半月板失稳）
- **动物**：70只雄性C57BL/6小鼠，8-10周龄
- **分组**：
  - Sham（假手术）
  - 5/6Nx-Sham（仅CKD）
  - DMM（仅OA）
  - 5/6Nx-DMM（CKD+OA共病）
- **终点**：12周，组织学评分、µCT、IHC、血清生化

### 核心发现

#### 1. 5/6Nx成功复制CKD特征

| 指标 | 5/6Nx-Sham vs Sham | 说明 |
|------|-------------------|------|
| 肾切除比例 | 70-80% | — |
| 体重增长 | ↓ | 系统性疾病 |
| 血清尿素 | **↑↑** | 肾功能↓ |
| 血清肌酐 | ↑ | — |
| 尿量 | ↑ | 浓缩功能↓ |
| 尿Cr/血Cr比值 | ↓ | — |
| **肾小管间质病变（TILs）** | **↑↑** | 纤维化+萎缩+扩张 |
| 血清ALP | **↑** | 骨代谢紊乱 |
| 血清磷 | **↑** | 磷潴留 |
| 血清钙 | **↑** | 继发性甲旁亢 |

#### 2. CKD单独不引起软骨病变，但改变骨微结构

| 指标 | 5/6Nx-Sham vs Sham | 意义 |
|------|-------------------|------|
| 软骨结构评分 | **无变化** | CKD不直接破坏软骨 |
| 蛋白聚糖丢失 | 无变化 | — |
| 滑膜炎评分 | 无变化 | — |
| **软骨下骨硬化** | **↓**（反而减少！） | 骨吸收主导 |
| **BV/TV** | **↓↓** | 骨量丢失 |
| **Tb.Th** | **↓** | 骨小梁变薄 |
| **Tb.N** | **↓** | 骨小梁数量↓ |
| **Tb.Sp** | **↑** | 骨小梁间隙↑ |
| **BMD** | **↓↓** | 骨密度↓ |
| **TRAP+破骨细胞** | **↑↑** | 骨吸收激活 |
| **滑膜MMP-13** | **↑** | **滑膜炎症/分解代谢** |
| **软骨MMP-13** | **无变化** | 组织特异性 |

**关键洞察**：CKD不直接攻击软骨，但通过**软骨下骨骨吸收** + **滑膜MMP-13↑**创造分解代谢环境。

#### 3. DMM单独诱导肾损伤（惊人发现！）

| 指标 | DMM vs Sham | 意义 |
|------|------------|------|
| **尿蛋白** | **↑** | 肾损伤标志 |
| **尿蛋白/肌酐比值** | **↑** | — |
| **uCr/pCr** | 改变 | — |
| **肾纤维化（TILs）** | **↑**（但<5/6Nx） | **OA可损伤肾！** |

**颠覆性发现**：OA→肾损伤是双向的！DMM手术本身（无NSAIDs）即可诱导肾纤维化。

#### 4. CKD+OA共病：双向修饰（核心结果）

| 指标 | 5/6Nx-DMM vs DMM | 方向 | 机制解释 |
|------|-----------------|------|---------|
| **软骨结构损伤** | **↓**（减轻） | **保护** | 骨吸收→负荷↓ |
| **蛋白聚糖丢失** | **↓** | **保护** | — |
| **骨赘成熟度/大小** | **↓↓** | **保护** | 骨吸收→骨赘形成↓ |
| 软骨细胞肥大 | 无变化 | — | — |
| 滑膜炎 | 无变化 | — | — |
| 软骨下骨硬化 | 无变化 | — | — |
| **BV/TV** | **↓↓** | 骨丢失 | CKD骨吸收持续 |
| **Tb.Th** | **↓** | 骨丢失 | — |
| **BMD** | **↓↓** | 骨丢失 | — |
| **破骨细胞（TRAP）** | **↓**（vs 5/6Nx-Sham） | 被DMM抑制 | DMM骨形成期 |
| 血清磷 | **↓**（vs 5/6Nx-Sham） | DMM影响 | 骨形成消耗磷 |
| 血清钙 | **↓**（vs 5/6Nx-Sham） | DMM影响 | 骨形成消耗钙 |
| 血清尿素 | **↑**（vs 5/6Nx-Sham） | DMM加重CKD | 分解代谢↑ |

### 机制解释

```
CKD单独:
    软骨下骨骨吸收↑（破骨细胞↑）
        ↓
    BV/TV↓ + BMD↓ + Tb.Th↓
        ↓
    软骨机械负荷↓（假性保护）
        + 滑膜MMP-13↑（分解代谢环境）
    
DMM单独:
    软骨损伤 + 软骨下骨硬化 + 骨赘形成
        + 滑膜炎 + 软骨MMP-13↑
        + 肾纤维化！（意外发现）

CKD+DMM:
    CKD骨吸收 对抗 DMM骨形成
        ↓
    净效应：软骨下骨质量↓ → 负荷缓冲↓ → 软骨应力↓
        ↓
    结果：软骨病变减轻（但骨质量更差！）
        + 骨赘形成↓（骨吸收抑制骨形成）
```

### 与临床的关联

| 临床现象 | 模型对应 |
|---------|---------|
| 透析患者OA更侵蚀性、骨赘少 | 模型中骨赘↓ |
| CKD患者骨折风险↑ | BV/TV↓ + BMD↓ |
| OA患者CKD风险↑ | DMM→肾纤维化 |
| NSAIDs加重CKD | 模型未用NSAIDs，纯OA即伤肾 |
| CKD-MBD改变骨质量 | 软骨下骨微结构重塑 |

### 局限性

- 仅10周观察期（人类是数年）
- 仅雄性小鼠
- 无肥胖/糖尿病等合并症
- "软骨保护"是骨吸收导致的机械减负，非真正的生物学保护
- 未检测FGF23、PTH、Vit D等CKD-MBD标志物

## 对Project-016的假说更新

### 假说v2：CKD-MBD→TMJOA的三层机制

```
第一层：系统性因素（CKD-MBD）
    ├─ FGF23↑（eGFR 60-90即升高）
    ├─ 磷潴留↑
    ├─ 继发性甲旁亢↑
    ├─ 活性Vit D↓
    ├─ 慢性炎症↑
    └─ 骨质量↓（高/低/混合型骨病）
        ↓
第二层：关节局部因素
    ├─ FGF23→FGFR1（Klotho非依赖）→ Wnt激活
    ├─ 软骨下骨骨吸收→微结构改变→负荷分布异常
    ├─ 滑膜MMP-13↑→分解代谢环境
    └─ 骨质量差→生物力学支撑↓
        ↓
第三层：TMJOA表型
    ├─ 软骨降解（MMP13/ADAMTS5/ColX↑）
    ├─ 软骨下骨硬化/囊变（重塑失衡）
    ├─ 骨赘形成（CKD中可能抑制）
    └─ 滑膜炎（MMP-13↑）
```

### 关键新增知识点

| 知识 | 来源 | 临床意义 |
|------|------|---------|
| FGF23Ab可挽救OA表型 | 文献1 | **FGF23中和抗体是潜在DMOAD** |
| Wnt/β-catenin是FGF23下游 | 文献1 | Wnt抑制剂有治疗潜力（全身毒性限制） |
| FGF23-Wnt正反馈环 | 文献1 | 打断任一节点均可 |
| CKD不直接破坏软骨 | 文献2 | CKD→OA是通过骨，不是直接软骨毒性 |
| 软骨下骨吸收"保护"软骨 | 文献2 | 假性保护，骨质量牺牲 |
| OA本身可损伤肾 | 文献2 | **OA→CKD双向性** |
| 滑膜MMP-13是CKD关节标志 | 文献2 | 可能比软骨MMP-13更敏感 |
