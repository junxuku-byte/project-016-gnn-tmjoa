# 动物实验为主 + 人体单臂验证：CKD-MBD→TMJOA治疗策略设计

> Project-016 | 临床转化路径v3 | 2026-05-16
> 核心翻转：从"人体RCT为主"→"动物机制为主 + 人体信号验证"

---

## 一、为什么这个策略更聪明

| 维度 | 人体RCT为主（原方案） | 动物为主+人体单臂（新方案） |
|------|---------------------|---------------------------|
| **机制证据** | 依赖文献外推（膝OA→TMJ） | **动物直接建立TMJ特异性因果链** |
| **样本量** | n=40×12月=高成本 | 动物n=8-10/组 + 人n=5-10 |
| **伦理难度** | 超适应症RCT难过 | 动物自由 + 人体观察性易过 |
| **时间** | 18-24个月 | **12-15个月** |
| **FGF23操控** | 不能（人只能测不能改） | **可以（过表达/敲低/抗体）** |
| **终点** | MRI评分未验证 | **动物可用组织学金标准** |
| **发表链** | 1篇RCT（难发高IF） | **2-3篇（机制→转化→临床信号）** |

**核心优势**：动物回答"FGF23是否因果驱动TMJOA？Denosumab是否保护TMJ？"，人体回答"这个发现在人身上能看到信号吗？"

## 二、动物实验设计（核心，70%权重）

### 2.1 模型选择

**问题**：没有现成的"CKD + TMJOA"双重模型，需要自建。

#### 方案A：叠加模型（推荐）

```
Step 1: 5/6肾切除 → CKD模型（文献2验证）
    ↓ 2周恢复期
Step 2: TMJOA诱导（三选一）
```

| TMJOA诱导方法 | 原理 | 优点 | 缺点 |
|-------------|------|------|------|
| **MIA注射** | 碘乙酸钠破坏软骨+滑膜 | 标准化、可重复、类似膝OA模型 | TMJ腔小，注射技术难 |
| **单侧前导（Unilateral Anterior Crossbite, UAC）** | 机械负荷异常→退变 | 模拟人类咬合因素，无药物干扰 | 手术复杂，周期长(8-12w) |
| **胶原酶+碘乙酸联合** | 酶消化+代谢毒性 | 退变快(4-6w) | 急性损伤，非慢性 |
| **DMM改编（TMJ盘失稳）** | 机械不稳定 | 类似37562858的DMM | 手术极精细，需要显微外科 |

**推荐**：大鼠MIA TMJ注射（技术成熟，文献多）或**小鼠UAC**（更模拟自然病程）。

**叠加时序**：
```
Week 0:  5/6Nx（或假手术）
Week 2:  TMJOA诱导（MIA注射或UAC手术）
Week 3:  分组给药
Week 6:  中期评估（n=3/组处死，确认模型成功）
Week 10-12: 终点评估
```

#### 方案B：转基因模型（如果合作方有）

```
HMWTg小鼠（文献1）→ 已有自发FGF23↑+膝OA
    ↓
额外检查TMJ（文献1没做TMJ！）
    ↓
若TMJ也有OA → 天然模型，只需给药
```

**风险**：HMWTg是膝OA模型，TMJ是否受累未知。需预实验。

### 2.2 分组设计

| 组别 | 处理 | n | 目的 |
|------|------|---|------|
| **1. Sham-Sham** | 假手术+假TMJ操作 | 8 | 基线对照 |
| **2. CKD-Sham** | 5/6Nx+假TMJ操作 | 8 | CKD单独对TMJ的影响 |
| **3. Sham-TMJOA** | 假肾手术+MIA/UAC | 8 | TMJOA单独 |
| **4. CKD-TMJOA** | 5/6Nx+MIA/UAC | 8-10 | **核心：CKD加重TMJOA？** |
| **5. CKD-TMJOA+Denosumab** | 5/6Nx+MIA+Denosumab | 8-10 | **核心干预** |
| **6. CKD-TMJOA+FGF23Ab** | 5/6Nx+MIA+FGF23Ab | 8-10 | **机制验证（如可及）** |

**Denosumab剂量**：参考骨质疏松模型 10mg/kg 皮下，每4周一次（换算自人体60mg Q6M）。

**FGF23Ab**：若不可及，可改用**BGJ398**（FGFR抑制剂，文献1用过，可购买）10mg/kg 口服。

### 2.4 样本量与统计

```
总动物：~50-60只（含预实验n=9）
每组n=8-10：检测能力（power）
    - 效应量（Cohen's d）= 1.0（大效应，预期明显）
    - α = 0.05
    - Power = 0.80
    - 每组仅需n=7，取n=8-10留余量

统计：
    - 多组比较：One-way ANOVA + Sidak/Tukey post-hoc
    - 两组比较：Student's t-test
    - 非参数备选：Kruskal-Wallis + Dunn's
    - 相关性：Pearson（FGF23 vs Mankin评分）
```

### 2.5 关键预期结果

| 假设 | 预期数据 | 若成立的意义 |
|------|---------|-------------|
| H1: CKD加重TMJOA | CKD-TMJOA vs Sham-TMJOA: Mankin↑30%, BV/TV↓20% | CKD是TMJOA独立危险因素 |
| H2: FGF23↑驱动TMJOA | 血清FGF23与Mankin正相关(r>0.4) | FGF23是生物标志物 |
| H3: Denosumab保护TMJ | Denosumab vs CKD-TMJOA: Mankin↓40%, BV/TV↑25% | **治疗靶点验证** |
| H4: FGF23-Wnt轴在TMJ激活 | 核β-catenin+、MMP13+、ColX+ | 机制与膝OA一致 |
| H5: Denosumab降FGF23 | 血清/软骨FGF23↓ | 间接降低FGF23驱动 |

## 三、人体单臂验证（辅助，30%权重）

### 3.1 设计逻辑

动物实验如果证明Denosumab保护TMJ→需要回答"人身上有信号吗？"
单臂设计：利用**已有骨质疏松适应症**的患者，收集TMJ数据作为**探索性终点**。

### 3.2 入组

```
单臂、开放标签、探索性
n = 8-12（小样本信号检测）

入组：
    ✓ 绝经后女性
    ✓ 骨质疏松（DXA T值 < -2.5）或骨量减少+高风险
    ✓ **合并TMJOA症状**（DC/TMD标准：关节痛/弹响/开口受限≥3月）
    ✓ 签署知情同意（写明探索性TMJ终点）

排除：
    ✗ TMJ手术史<6月
    ✗ 口腔感染/计划拔牙<6月
    ✗ eGFR<30
    ✗ 低钙血症
    ✗ 长期使用双膦酸盐

干预：
    Denosumab 60mg 皮下 Q6M（标准骨质疏松治疗）
    + 标准TMJOA保守治疗（口腔矫治器+NSAIDs按需）
```

### 3.3 终点

| 类型 | 指标 | 时间点 | 意义 |
|------|------|--------|------|
| **主要** | 血清FGF23变化 | 基线→3月→6月 | 靶点调制 |
| **次要1** | TMJ功能评分变化 | JFLS-20 + VAS | 临床信号 |
| **次要2** | 最大开口度变化 | 临床测量 | 功能改善 |
| **次要3** | 血清CTX/P1NP变化 | 基线→6月 | 骨转换标志物 |
| **探索** | TMJ MRI（可选n=3-5） | 基线→6月 | 影像信号（太贵可省） |
| **安全** | 血清钙、下颌骨症状 | 每次随访 | ONJ监控 |

### 3.4 成功标准（探索性，非统计效力）

```
若≥6/10例出现以下之一：
    - 血清FGF23下降>20%
    - VAS疼痛下降>30%
    - 开口度增加>10%
    - JFLS-20改善>MCID（最小临床重要差）

→ 视为"人体验证信号阳性"，支持启动正式RCT
```

## 七、风险与对策

| 风险 | 概率 | 对策 |
|------|------|------|
| 5/6Nx死亡率高 | 中 | 改良手术（单侧肾切除+对侧动脉结扎），降低死亡率 |
